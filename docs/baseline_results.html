Baseline Results

LumbarSR: A Paired Clinical CT and Photon-Counting Micro-CT Dataset for Human Lumbar Vertebrae

Experimental Setup

  • Training Set: Lumbar_01 to Lumbar_25 (25 samples)
  • Test Set: Lumbar_26 to Lumbar_30 (5 samples)
  • Training: Deep learning methods trained on 25 samples with single-sequence soft-kernel input by default; optional paired-kernel input uses the B + S pair from the same FOV
  • Evaluation: Metrics computed as mean ± standard deviation across test set
  • Baseline: Registered clinical CT without super-resolution processing

Key Findings

  • SRCNN achieves the best PSNR with 8-9% improvement over baseline
  • UNet excels at structural preservation with the highest SSIM scores
  • Interpolation methods perform identically to baseline (input/output dimensions match after registration)
  • BoneMask-ROI evaluation shows more discriminating metrics than full-image evaluation
  • Deep learning methods show consistent improvements in both Small and Large FOV configurations

Reference Aggregate Summary

How The Summary Is Computed

Step 1: For each method, compute one aggregate score per metric by averaging across all FOV and window combinations:

Metric Category Aggregate Score Direction
PSNR Score Mean of 4 PSNR values (2 windows × 2 FOV, BoneMask ROI) ↑ Higher is better
SSIM Score Mean of 4 SSIM values (2 windows × 2 FOV, BoneMask ROI) ↑ Higher is better
MAE Score Mean of 4 MAE values (2 windows × 2 FOV, BoneMask ROI) ↓ Lower is better
LBC Score Mean LBC Ratio across all input sequences ↑ Closer to 1.0 is better

Step 2: For each metric, methods are ordered independently and the four orders are averaged.

Reference aggregate = (PSNR_Order + SSIM_Order + MAE_Order + LBC_Order) / 4
This value is intended only as a compact cross-metric summary. The metric-specific tables above remain the primary results for interpretation.
Example: If one method is ordered 1st in PSNR, 2nd in SSIM, 1st in MAE, and 3rd in LBC, its reference aggregate is (1+2+1+3)/4 = 1.75.

Methods Description

  • Baseline: Registered clinical CT without any super-resolution processing
  • UNet: U-shaped encoder-decoder architecture with skip connections (~30M parameters)
  • SRCNN: Super-Resolution Convolutional Neural Network (~60K parameters)
  • Nearest: Nearest neighbor interpolation (represents all interpolation methods)
