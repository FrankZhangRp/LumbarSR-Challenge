Baseline Results

Lumbar Vertebral Super-Resolution Challenge

Experimental Setup

  • Training Set: Lumbar_01 to Lumbar_25 (25 samples)
  • Test Set: Lumbar_26 to Lumbar_30 (5 samples)
  • Training: Deep learning methods trained on 25 samples with dual-channel input (small FOV + large FOV)
  • Evaluation: Metrics computed as mean Â± standard deviation across test set
  • Baseline: Registered clinical CT without super-resolution processing

Key Findings

  • SRCNN achieves the best PSNR with 8-9% improvement over baseline
  • UNet excels at structural preservation with the highest SSIM scores
  • Interpolation methods perform identically to baseline (input/output dimensions match after registration)
  • Masked evaluation shows more discriminating metrics than full-image evaluation
  • Deep learning methods show consistent improvements in both Small and Large FOV configurations

Ranking System

How Rankings Are Computed

Step 1: For each team, compute one aggregate score per metric by averaging across all FOV and window combinations:

Metric Category Aggregate Score Direction
PSNR Score Mean of 4 PSNR values (2 windows Ã— 2 FOV, masked) â†‘ Higher is better
SSIM Score Mean of 4 SSIM values (2 windows Ã— 2 FOV, masked) â†‘ Higher is better
MAE Score Mean of 4 MAE values (2 windows Ã— 2 FOV, masked) â†“ Lower is better
LBC Score Mean LBC Ratio across all input sequences â†‘ Closer to 1.0 is better

Step 2: For each metric, all teams are ranked independently (Rank 1 = best, Rank 2 = second best, ...).

Final Score = (PSNR_Rank + SSIM_Rank + MAE_Rank + LBC_Rank) / 4
Lowest average rank wins. LBC Rank breaks ties.
Example: If a team ranks 1st in PSNR, 2nd in SSIM, 1st in MAE, 3rd in LBC â†’ Final Score = (1+2+1+3)/4 = 1.75
